# Resume Matcher & Builder

A powerful MERN stack application that helps users create professional resumes and optimize them for Applicant Tracking Systems (ATS) using AI.

## Features

- **AI Resume Analysis**: Get instant feedback on your resume's compatibility with job descriptions.
- **Resume Builder**: Create professional resumes with multiple templates.
- **ATS Optimization**: AI-driven suggestions to improve your ATS score.
- **Interview Prep**: Generate custom interview questions based on your resume and target job.
- **Cover Letter Generator**: Create tailored cover letters in seconds.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express, MongoDB
- **AI**: Google Gemini 1.5 Flash

## Setup

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    cd backend && npm install
    cd ../frontend && npm install
    ```
3.  Set up environment variables in `backend/.env` and `frontend/.env`.
4.  Run the development servers:
    ```bash
    # Backend
    cd backend && npm run dev

    # Frontend
    cd frontend && npm run dev
    ```
